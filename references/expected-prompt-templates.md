# Expected Prompt Templates

Use these Chinese prompt templates as the default wording pattern when executing the workflow. Fill placeholders from the approved source material, outline, production pack, style contract, and slide job.

## 1. Outline Agent Prompt

```text
现在我给你提供了一份文件/代码/论文/实验记录/文本材料。
我要根据这些内容制作一份PPT，并用于汇报。请先深入分析我提供的材料，严格基于材料内容进行总结，不要自行编造。

要求：
1. PPT页数控制在 <页数上限> 页以内；
2. 请提供整套PPT的大纲；
3. 请给出每一页的标题；
4. 每一页标题要服务于汇报逻辑；
5. 不要脱离我提供的原始材料；
6. 对源材料没有提供的数据、结论、实验结果或引用，必须标注为“材料未提供”，不得补写。

输出：
- PPT标题
- 汇报逻辑
- 页数
- 每页标题
- 每页角色
- 每页是否需要插图或真实实验图
```

## 2. Per-Slide Content Agent Prompt

```text
现在我已经确定了这份PPT的大纲和标题。下面我会提供：
1. 原始材料；
2. 已确定的PPT大纲；
3. 每一页标题。

请严格参考原始材料和大纲，给出每一页PPT应该填充的具体内容。

要求：
1. 每页内容要服务于该页标题；
2. 内容适合放在PPT中，不要写成大段论文；
3. 每页给出页面核心观点、要点说明和建议插图；
4. 不要脱离原始材料；
5. 不要自行编造数据、结论或实验结果；
6. 如果某页需要真实实验图、数据图或截图，请明确标为 strict input asset；
7. 如果某页只有风格参考图，请明确标为 style reference。

每页输出格式：
第 <页码> 页：<标题>
### 页面核心观点
...
### 页面内容
- ...
- ...
- ...
### 建议插图
...
### 来源依据
...
### 不确定或缺失信息
...
```

## 4. First Image2 Slide Prompt

Use this for the first generated sample or the first slide that uses a user-provided PPT reference image.

```text
必须调用 Image2/Nano2 风格的完整页面生图能力。我要制作一页 16:9 比例的中文PPT页面。

我已经提供了两类材料：
1. 一张已制作好的PPT参考图，用于参考整体风格；
2. 当前这一页PPT的标题和内容。

请根据参考图的视觉风格进行设计，但不要机械照搬排版。要求整体风格保持连贯，同时在版式上根据本页内容做合理创新。

当前页标题为：
<严格使用已批准标题，不要修改>

当前页主要内容为：
<来自生产包的核心观点和页面内容>

建议插图 / 主视觉：
<SCI风格插图、流程图、结构图、对比图、结果图等>

设计要求：
1. 内容必须严格参考我提供的材料，不得自行编造；
2. 标题不需要修改，只需围绕标题填充页面内容；
3. 页面要图文并茂，不能只有文字；
4. 插图要符合SCI论文插图风格，逻辑合理，能够准确表达本页内容；
5. 插图要求清晰、线条精细、矢量图质感；
6. 整体画面要求学术、清晰、精细，具有4K级视觉质感；
7. 中文字体统一使用黑体风格，不能出现字体变形；
8. 所有中文必须无乱码、无错字、无扭曲；
9. 页面排版要规整，不能拥挤，不能拉伸变形；
10. 生成完成后必须检查中文字体、页面比例、内容准确性和插图合理性；
11. 不要添加水印、无关logo、随机页码或材料中没有的信息。
```

## 6. Strict Experiment/Data Figure Prompt Block

Append this block whenever a slide includes a real experiment figure, data plot, real screenshot, table, or result chart.

```text
重要：我上传的实验图/数据图是真实结果图，只能作为插图放入页面。
只能对它进行等比例放大或缩小，不能拉伸，不能变形，不能重画，不能修改其中的任何文字、曲线、坐标、图例、颜色、标签和数值。
不要把真实结果图重新生成成插画，不要翻译图中文字，不要平滑曲线，不要改变数据点，不要改变坐标轴和图例。
如果无法保证原图内部内容不被AI修改，请停止该页生图，并改为预留版面或使用机械放置方式。
```

## 7. Per-Slide QA Prompt

```text
请检查刚生成的这一页PPT图：
1. 是否为16:9比例；
2. 中文是否无乱码、无错字、无扭曲；
3. 标题是否保持原题，不被修改；
4. 页面内容是否严格来自原始材料；
5. 是否出现材料中没有的数据、结论或实验结果；
6. 插图是否符合SCI论文插图风格，且能表达本页内容；
7. 排版是否规整，不拥挤，不拉伸变形；
8. 如果包含真实实验图或数据图，其内部文字、曲线、坐标、图例和数值是否未被修改；
9. 是否与参考风格图或上一页风格保持连贯。

如果任一项失败，请指出失败原因并重新生成该页，不要继续生成下一页。
```
